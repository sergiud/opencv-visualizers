OpenCV debugger visualizers for Visual Studio
=============================================

:Author: Sergiu Dotenco
:Date:   March 13, 2012

Getting Started
---------------

To install Visual Studio 2005 and 2008 visualizers, either create an environment
variable called ``_vcee_autoexp`` and let it point to ``autoexp.dat``, or copy
the content of ``autoexp.dat`` to ``<Visual Studio installation
directory>\Common7\Packages\Debugger\autoexp.dat``.
